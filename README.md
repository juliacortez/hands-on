<h2>⚛️ﾠSobre</h2>
Projeto frontend realizado para o desafio Hands On da <a href="https://tracepack.com.br/">Trace Pack.</a> Seguindo os requisitos propostos o usuário encontra logo de início uma página de login, que ao ser realizado dá acesso a visualização de um mapa interativo. Neste mapa é possível que o usuário crie um ponto ao clicar no mapa, e dê um nome ao mesmo, que é exibido em um pop-up. Além da possbilidade de criar polígonos, nomear e mudar a cor da forma. 

<h2>🛠️ Tecnologias</h2>
<li><a href="https://pt-br.reactjs.org/">React.js</a> (v17.0.2)</li>
<li><a href="https://styled-components.com/docs">Styled-components</a> (v5.3.3)</li>
<li><a href="https://reactrouterdotcom.fly.dev/docs/en/v6/getting-started/overview">React Router</a> (v6)</li>
<li><a href="https://react-leaflet.js.org/">React-Leaflet</a> (v5.5.2)</li>


<h2>💻 Pré-requisitos</h2>
Antes de começar você vai precisar ter instalado em sua máquina as seguintes ferramentas:
<a href="https://git-scm.com">Git</a>, <a href="https://nodejs.org.en/">Node.js</a>.
Além disto ter um editor para trabalhar com o código, para o desenvolvimento deste projeto foi utilizado o <a href="https://code.visualstudio.com/">VSCode</a>.

<h2>🚀 Rodando o projeto</h2>

```bash
# Clone este repositório
# HTTPS
$ git clone https://github.com/juliacortez/hands-on.git

# CLI
$ gh repo clone juliacortez/hands-on

# Acesse a pasta do projeto no terminal/cmd
$ cd hands-on

# Instale as dependências
$ npm install

# Execute o comando para iniciar a aplicação
$ npm run start

# O servidor iniciará na porta:3000 - acesse <http://localhost:3000>
```
